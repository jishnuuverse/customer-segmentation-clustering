# Customer Segmentation using Clustering

## Overview
This project uses K-Means and DBSCAN clustering to segment customers based on income and spending behavior.

## Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Features
- Data preprocessing and scaling
- K-Means clustering
- Elbow method for optimal clusters
- Silhouette score evaluation
- DBSCAN clustering for outlier detection

## Dataset
Mall Customers Dataset

## Output
Customer groups based on behavior for business insights
